• Tables

    A very common task in HTML is structuring tabular data, and it has a number of elements and attributes for just this purpose. Coupled with a little CSS for styling, HTML makes it easy to display tables of information on the web such as your school lesson plan, the timetable at your local swimming pool, or statistics about your favorite dinosaurs or football team. This module takes you through all you need to know about structuring tabular data using HTML.

  • What is table?

    A table is a structured set of data made up of rows and columns (tabular data). A table allows you to quickly and easily look up values that indicate some kind of connection between different types of data, for example a person and their age, or a day of the week, or the timetable for a local swimming pool.

  • Structure Of The Table

    All the table content we should put inside the <table> tag. Just like <body> tag for all visible content we use <table> for all tabular data inside it. Then we use <tr> (stands for table row) tag and inside <tr> we use <td> (stands for table data). This works similar to <ul> and <li> structure.
    <table>
    ㅤㅤ<tr>
    ㅤㅤㅤㅤ<td>I am the firt cell</td>
    ㅤㅤㅤㅤ<td>I am the second cell</td>
    ㅤㅤㅤㅤ<td>I am the third cell</td>
    ㅤㅤ</tr>
    ㅤㅤ<tr>
    ㅤㅤㅤㅤ<td>I am the firt cell, second row</td>
    ㅤㅤㅤㅤ<td>I am the second cell, second row</td>
    ㅤㅤㅤㅤ<td>I am the third cell, second row</td>
    ㅤㅤ</tr>
    </table>
    I used a some CSS to make black borders and separate the data. Without CSS it would be unreadable and look like some texts with texts under it.

  • Table Headers

    Now let's turn our attention to table headers — special cells that go at the start of a row or column and define the type of data that row or column contains. We use <th> to use them. Lets add <th>s to our example above.

    ㅤㅤ<tr>
    ㅤㅤㅤㅤ<th>I am the firt Row Header</th>
    ㅤㅤㅤㅤ<td>I am the firt cell</td>
    ㅤㅤㅤㅤ<td>I am the second cell</td>
    ㅤㅤㅤㅤ<td>I am the third cell</td>
    ㅤㅤ</tr>
    ㅤㅤ<tr>
    ㅤㅤㅤㅤ<th>I am the second Row Header</th>

    ㅤㅤㅤㅤ<td>I am the second cell, second row</td>
    ㅤㅤㅤㅤ<td>I am the third cell, second row</td>
    ㅤㅤ</tr>
    </table>
