• HTML Structure

    In this article we'll learn the main structure of every website. This course contains the most important: tags html, Body, Head and Footer and some other elements.

  • HTML Tag

    The <html> element represents the root (top-level element) of an HTML document, so it is also referred to as the root element. All other elements must be descendants of this element. Basically you'll just cover your whole html document into this tag and add the language element. It should look like this:
    <html lang="en">
    --Your Code Goes Here--
    </html>

  • Head

    The Head of an HTML document is the part that is not displayed in the web browser when the page is loaded. It contains information such as the page <title>, links to CSS (if you choose to style your HTML content with CSS), links to custom favicons, and other metadata (data about the HTML, such as the author, and important keywords that describe the document). Web browsers use information contained in the head to render the HTML document correctly.

    example of typical <head> :
    <head>
    ㅤㅤ<meta charset="UTF-8">
    ㅤㅤ<title>PageTitle</title>
    ㅤㅤ<link rel="stylesheet" href="YourCSS.css">
    </head>
    Now, lets take a look at each element.
    1. <meta> represents Metadata that cannot be represented by other HTML meta-related elements. charset attribute declares the document's character encoding. If the attribute is present, its value must be an ASCII case-insensitive match for the string "utf-8", because UTF-8 is the only valid encoding for HTML5 documents.
    2. <title> element defines the document's title that is shown in a Browser's title bar or a page's tab. It only contains text; tags within the element are ignored.
    3. <link> specifies relationships between the current document and an external resource. This element is most commonly used to link to CSS, but is also used to establish site icons (both "favicon" style icons and icons for the home screen and apps on mobile devices) among other things. The rel element tells what type of data the document contains and the href element tells us the documents's url, just like in <a> tag

  • Body

    The <body> element represents the content of an HTML document. There can be only one <body> element in a document. Basically, you put all the visible content here. Here is example:
    <body>
    ㅤㅤ<p>I love oranges</p>
    </body>

  • Footer

    The <footer> element represents a footer for its nearest ancestor sectioning content or sectioning root element. A <footer> typically contains information about the author of the section, copyright data or links to related documents, for example:
    <footer>
    ㅤㅤ<p>© 2022 Vyo</p>
    </footer>

  • Summary

    Now lets put every element together to get a complete structured website:
    <html lang="en">

    ㅤㅤ<head>
    ㅤㅤㅤㅤ<meta charset="UTF-8">
    ㅤㅤㅤㅤ<title>PageTitle</title>
    ㅤㅤㅤㅤ<link rel="stylesheet" href="YourCSS.css">
    ㅤㅤ</head>

    ㅤㅤ<body>
    ㅤㅤㅤㅤ<p>I love oranges</p>
    ㅤㅤ</body>

    ㅤㅤ<footer>
    ㅤㅤㅤㅤ<p>© 2022 Vyo</p>
    ㅤㅤ</footer>

    </html>
