• What's next

    If you are reading this article, chances are you already read all the modules above and know html basics. This page will contain all the possible ways to continue your study in web development. I will give links to some of the teaching sites/videos i can recommend here.

  • CSS

    The first thing you should explore after HTML is CSS. You may remember i mentioned this language in some modules before. Basically CSS(Cascading Style Sheets) is a style sheet language used for describing the presentation of a document written in a markup language such as HTML or XML. CSS is designed to enable the separation of presentation and content, including layout, colors, and fonts.

    To learn it i would recommend using MDN Web Docs. Put in some hard work in you can learn it in a couple weeks.

  • JavaScript

    To master the FrontEnd web development you should learn JavaScript, often abbreviated as JS, is a programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS. As of 2022, 98% of websites use JavaScript on the client side for webpage behavior, often incorporating third-party libraries. All major web browsers have a dedicated JavaScript engine to execute the code on users' devices.

    To learn it i would still recommend MDN Web Docs. However, JS is a complete programming language and is really hard to learn. Alongside with JS you'll need to learn frameworks that can be used with it, for example: Node.js, React.js.

  • BackEnd

    Visual part of the saide is much easier then its mechanism behind the scenes. However for backend variety of languages is larger. You can use Java, PHP, Python, C# and some more. You should think some before choosing a language because picking your language is really important and difficult.

  • But overall all you need to do is Practice. Practice makes perfect. Always create something, google something, learn something, write something.
